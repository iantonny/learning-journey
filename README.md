### ✅ Week 2 — Arrays & Strings

**Concepts Learned:**
- Arrays — storing multiple values efficiently
- Strings as arrays of characters
- strlen — measuring string length
- toupper and tolower — character manipulation
- isupper and islower — character checking
- Character arithmetic — converting letters to array positions
- Command line arguments — argc and argv
- Optimized for loops — declaring multiple variables
- Global arrays and constants
- Defensive programming — handling unexpected user input
- Coleman-Liau index — readability formula
- Caesar cipher — encryption using modular arithmetic
- atoi — converting string to integer
- isdigit — validating numeric input
- Modulo arithmetic — wrapping numbers within a range

**Practice Programs:**
| Program | Description |
|---------|-------------|
| average.c | Calculates average of scores using arrays and functions |
| string_length.c | Measures string length and prints each character |
| uppercase.c | Converts string to uppercase using toupper |
| uppercase_v2.c | Optimized version using single for loop declaration |
| commandline.c | Accepts and validates command line arguments |

**Key Insight From Week 2:**
Arrays make programs faster and smarter.
Storing 1000 values in one place instead of 1000 separate variables.
This is exactly how AI handles massive datasets.
